# 一条命令启动项目

在项目根目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

脚本会自动完成这些事：

- 如果 `config.json` 不存在，生成一个本地开发用的最小配置。
- 如果 `credentials.json` 不存在，生成空数组，后续可在 Admin UI 里添加凭据。
- 如果 `admin-ui/dist` 不存在，自动安装前端依赖并构建 Admin UI。
- 启动后端服务：`cargo run -- --config config.json --credentials credentials.json`。

启动成功后访问：

```text
http://127.0.0.1:8990/admin
```

本地配置、凭据、客户端 Key、日志、数据库和构建产物都已在 `.gitignore` 中排除，不会上传到 GitHub。正式环境拉取代码后，保留正式环境自己的 `config.json` 和 `credentials.json` 即可。

## 常用命令

本地开发完成后上传到 GitHub：

```powershell
git add .
git commit -m "update"
git push
```

正式环境拉取更新：

```bash
git pull origin master
```
